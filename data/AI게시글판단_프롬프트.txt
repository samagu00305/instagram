**중요: 실제 참여 가능한 이벤트인지 판단해줘**

**1단계 - 이벤트 여부 판단:**
게시물의 내용과 목적을 파악해서, 다음 중 하나라도 해당한다고 판단되면 무조건 '바로 참여 여부'를 '불가'로 써줘:
- 외국어 게시물 (영어, 일본어, 중국어 등 한국어가 아닌 언어로 주로 작성된 글)
- 단순 홍보 목적 (식당/카페 맛집 추천, 상품 소개, 할인 정보 안내만 하는 글)
- 방문 유도만 하는 글 (이벤트 없이 그냥 "저장하고 와봐", "한번 방문해봐" 같은 권유만 있는 글)
- 전시회/박람회/설명회/축제/행사 참관 유도 글 (경품 없이 현장 방문만 권유하는 글)
- "오세요", "방문하세요", "구경하세요", "즐기러 오세요" 같은 단순 방문 유도 문구만 있는 글
- 주소나 위치 정보만 제공하는 홍보성 글
- 체험단/서포터즈/리뷰어 모집 (선발 과정이 필요한 것)
- 제품 판매나 쇼핑몰 광고 글

진짜 이벤트는: 한국어 게시물 + 명확한 참여방법 + 기간 + 경품이 있어야 함

**중요: 경품이 없으면 무조건 이벤트가 아님**
아무리 참여 방법이 있어도 경품/당첨/상품/증정 내용이 명확하지 않으면 '바로 참여 여부'를 '불가'로 써줘.

**이벤트가 아닌 경우:**
위 조건에 해당해서 '바로 참여 여부'가 '불가'라면:
- 끝 날짜: 해당없음
- 진행상태: 해당없음
- 바로 참여 여부: 불가
- 제외 판단: 아니오
이렇게 출력하고 더 이상 판단하지 마.

**2단계 - 실제 참여 방법 확인:**
진짜 이벤트라면 다음이 **명확하게 텍스트로** 명시되어야 함:
- 구체적 참여 방법 (예: "댓글로 OO을 남겨주세요", "팔로우 후 좋아요 눌러주세요")
- 이벤트 기간 또는 마감일
- 경품/당첨 내용 (예: "추첨을 통해 OO을 드립니다")

**중요: 위 3가지 중 하나라도 명확하게 적혀있지 않으면 '바로 참여 여부'를 무조건 '불가'로 써줘.**
"오세요", "구경하세요" 같은 단순 권유는 참여 방법이 아님.

**중요: 참여 방법이 없으면 여기서 중단**
게시물에 "댓글 달아주세요", "좋아요 눌러주세요", "팔로우 해주세요" 같은 명확한 참여 방법이 하나도 없으면 '바로 참여 여부'를 '불가'로 쓰고 더 이상 판단하지 마.

**중요: 참여 방법 제한**
참여 가능한 간단한 작업 (이것들만 있으면 OK):
- 댓글 달기
- 좋아요 누르기
- 팔로우 하기

참여 불가 (추가 작업):
- 친구 태그 (@친구아이디 태그하기)
- 스토리 공유/리그램
- 릴스 업로드
- 게시물 저장 요구
- 외부 링크 클릭 (프로필 링크, 바이오 링크, URL 접속 등)
- 외부 앱/사이트에서 작업 (설정, 다운로드, 가입, 구매 등)
- 기타 추가 작업

판단 기준:
**게시물을 읽고 이벤트 참여 방법이 무엇인지 판단해라**

질문: "이 게시물의 이벤트 참여 방법은 무엇인가?"
- 게시물을 읽고 사용자가 어떤 행동을 해야 이벤트에 참여하는 것인지 판단해라
- 참여 방법이 명확하게 적혀있지 않으면 → 무조건 불가
- **중요: 선택사항 무시하기**
  - "당첨 확률 UP", "당첨 확률 증가", "선택사항", "추가로 하면" 같은 표현이 있으면 그건 필수가 아님
  - 필수 참여 방법만 판단해라

판단한 참여 방법이:
- "댓글 달기"만 요구하는가? → 가능
- "좋아요 누르기"만 요구하는가? → 가능
- "팔로우 하기"만 요구하는가? → 가능
- "댓글+좋아요" 조합만 요구하는가? → 가능
- "댓글+좋아요+팔로우" 조합만 요구하는가? → 가능

판단한 참여 방법에 아래 중 하나라도 포함되어 있는가? → 무조건 불가
- 친구 태그하기 (@친구아이디 태그)
- 스토리 공유하기 / 리그램
- 릴스 업로드하기
- 게시물 저장하기
- 외부 링크 클릭 (프로필 링크, 바이오 링크, 링크 타고, URL 접속)
- 외부 앱/사이트에서 작업 (앱 설정, 다운로드, 가입, 영상 설정 등)
- 기타 추가 작업

**제외 판단:**
게시물의 내용과 맥락을 보고 다음 중 하나에 해당한다고 판단되면 '제외 판단'을 '예'로 써줘:
- 당첨자 발표 글 (이미 이벤트가 끝나고 당첨자를 안내하는 글)
- 설명회나 특정 대상자만 참여 가능한 모집 글
- 사전 신청이 필요했던 이벤트 (이미 사전 신청 기간이 지남)
- 인원 모집 글 (단, 경품 당첨 인원 수를 말하는 건 제외)
위 경우가 아니라고 판단되면 '제외 판단'을 '아니오'로 써줘.

**바로 참여 여부 판단:**
이벤트가 맞다면 다음을 판단해서 '바로 참여 여부'를 결정해줘:
- 참여기간 끝 날짜를 확인하고, 진행중이면 진행중, 기간 지났으면 종료라고 써줘
- 참여 기간이 지났으면 '바로 참여 여부'를 무조건 '불가'로 써줘
- 외부 링크 클릭이나 외부 앱/사이트 작업이 필요하면 '불가'
- 참여방법이 댓글, 좋아요, 팔로우처럼 인스타그램 안에서만 가능하면 '가능'
- 다른 형태이거나 추가 작업이 필요하면 '불가'
- 당첨 상품이 할인권, 할인쿠폰, 구매 링크처럼 추가 구매가 필요하다고 판단되면 '불가'

**경고: 아래 형식 그대로 출력하고 다른 말은 절대 하지 마:**

끝 날짜 - (날짜 또는 '해당없음')
진행상태 - (진행중/종료/해당없음 중 하나)
바로 참여 여부 - (가능/불가 중 하나)
제외 판단 - (예/아니오 중 하나)

**출력 예시:**
끝 날짜 - 11월 30일
진행상태 - 진행중
바로 참여 여부 - 가능
제외 판단 - 아니오

**중요: "끝 날짜 -", "진행상태 -" 같은 라벨을 절대 빼지 마. 위 형식 그대로 출력.**

**위 4줄만 출력. 인사말, 설명, '판단합니다', '바탕으로' 같은 추가 문구 절대 금지. 오직 위 4줄만.**
