# 인스타그램 이벤트 자동화

store 프로젝트를 참고하여 만든 인스타그램 이벤트 자동화 프로그램입니다.

## 폴더 구조 11223344

```
instagram/
├── Main.ahk              # 메인 실행 파일
├── Util.ahk              # 유틸리티 함수 (이미지 인식, 클릭 등)
├── GlobalData.ahk        # 전역 변수 및 상수
├── Gdip_All.ahk          # GDI+ 라이브러리
├── Image/                # 이미지 저장 폴더
│   └── home/             # 홈 화면 관련 이미지
│       └── home.png      # 여기에 홈 화면에서 클릭할 이미지를 넣으세요
└── README.md             # 이 파일
```

## 사용 방법

### 1. 홈 화면 이미지 준비
1. 인스타그램 홈 화면에서 클릭하고 싶은 부분을 캡처합니다
2. 캡처한 이미지를 `Image/home/home.png`로 저장합니다
3. 이미지는 PNG 형식이어야 합니다

### 2. 프로그램 실행 (VSCode에서만)

#### 방법 1: F5 키로 실행 (가장 간단!) ⭐
1. VSCode에서 `Main.ahk` 파일을 엽니다
2. **F5 키**를 누릅니다
3. 끝!

#### 방법 2: Run Code 버튼 (▶️)
1. VSCode에서 `Main.ahk` 파일을 엽니다
2. 우측 상단의 **▶️ Run Code** 버튼을 클릭합니다
3. 또는 `Ctrl + Alt + N` 단축키를 누릅니다

#### 방법 3: 터미널 명령어
VSCode 터미널(PowerShell)에서:
```powershell
& 'C:\Program Files\AutoHotkey\AutoHotkey.exe' .\Main.ahk
```

### 3. 테스트
1. 인스타그램 화면을 띄워놓습니다
2. "홈 화면 이미지 클릭 테스트" 버튼을 클릭합니다
3. 프로그램이 자동으로 이미지를 찾아서 클릭합니다

## 주요 기능

### 현재 구현된 기능
- ✅ 홈 화면 이미지 찾기 및 클릭
- ✅ 이미지 인식 기반 자동화
- ✅ GUI 인터페이스

### 추후 구현 예정
- ⏳ 인스타그램 앱 자동 실행
- ⏳ 이벤트 자동 참여
- ⏳ 댓글 자동 작성
- ⏳ 좋아요 자동 클릭

## 주요 함수 설명

### Util.ahk
- `FindImage_Byref()`: 화면에서 이미지 검색
- `ClickAtWhileFoundImage()`: 이미지를 찾아서 클릭
- `WhileFoundImage()`: 이미지를 찾을 때까지 대기
- `MoveAtWhileFoundImage()`: 이미지를 찾아서 마우스 이동
- `IsImageExist()`: 이미지 존재 여부 확인

## 이미지 인식 팁

1. **명확한 이미지 사용**: 배경과 구분이 명확한 이미지를 사용하세요
2. **적절한 크기**: 너무 크거나 작지 않은 적절한 크기로 캡처하세요
3. **고유한 패턴**: 화면에서 유일하게 구분되는 부분을 캡처하세요

## 문제 해결

### 이미지를 찾을 수 없습니다
- `Image/home/home.png` 파일이 존재하는지 확인하세요
- 인스타그램 화면이 화면에 보이는지 확인하세요
- 이미지가 화면의 해상도와 일치하는지 확인하세요

### 클릭이 정확하지 않습니다
- `ClickAtWhileFoundImage()` 함수의 `addX`, `addY` 파라미터를 조정하세요
- 예: `ClickAtWhileFoundImage("home\home", 10, 10)` → 이미지 위치에서 오른쪽 10px, 아래 10px 이동
